Variables: var

Portée: la fonction courante


Utilisation : 

var maVar [= maValeur];

Variables: let

Portée: le bloc courant ( {...} )


Utilisation : 

let maVar [= maValeur];

Variables: const

Permet de créer une constante qui ne peut plus être réaffectée. La valeur n'est pas immuable

Portée: la fonction courante


Utilisation : 

const maVar [= maValeur];

Hoisting

Toute déclaration de variable ou fonction déclarée, est remontée en haut de sa portée.

L'affectation se fait là où la variable est définie.

if / else

Permet d'exécutee une instruction si une condition donnée est vraie. En cas contraire ça passe dans le else

if (condition) {
  instruction1
} [else if (autre condition){
  instruction2
}] [else {
  instruction3
}]

condition est une expression qui est évaluée à true ou false.

for

Permet de créer une boucle.

for ([initialisation]; [condition]; [expression_finale]) {
  instruction
}

Exemple: on itère 10 fois (de 0 à 9) 

var i;
for (i = 0; i <= 9; i++) {
  console.log(i);
}

while

Permet de créer une boucle s'éxécutant tant qu'une condition est vrai.

while (condition) {
  instruction
}

Exemple: on itère tant que n est inférieur à 3 

var i = 0;
while (i < 3) {
  console.log(i);
  i++;
}

do ... while

Permet de créer une boucle s'éxécutant tant qu'une condition est vrai et s'éxécutant au moins une fois.

do {
  instruction
} while (condition)

Exemple: on itère tant que n est inférieur à 3 

var i = 0;
do {
  console.log(i);
  i++;
} while (i < 3)

try / catch

Permet de regrouper un ensemble d'instructions dans le try et de gérer les erreurs si une instruction provoque une exception. 

try {
  instructions_try
}
[catch (exception_var_1) {
  instructions_catch_1
}]
...
[catch (exception_var_2) {
  instructions_catch_2
}]
[finally {
  instructions_finally
}]
